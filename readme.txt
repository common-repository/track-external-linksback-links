== Installation ==
This section describes how to install the plugin and get it working.

1.Upload the entire Back link tracker  Plugin folder to the /wp-content/plugins/ directory.
2.Activate the plugin through the \'Plugins\' menu in WordPress.
3.You will find \'Backlink Tracker\' menu in your WordPress admin panel.
